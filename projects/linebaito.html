LINE Baito

In spring 2020, I participated in LINE Corporation’s competitive UI Design Internship Course, where I independently redesigned the user interface of LINE Baito, the company’s part-time job search and recruitment service app targeted at students and young workers in Japan.

I redesigned the UI for LINE Baito with a focus on usability, visual hierarchy, and mobile-first user flow. The redesign covered key features like the job feed, filtering, and application process, aligning with LINE’s design system to reduce cognitive load and improve accessibility.

Using Adobe XD, I built a static prototype incorporating real-world use cases, such as job search by location and LINE-based applications. I presented the final design to LINE’s product team and recruiters, receiving feedback on user flow and integration ideas like LINE Pay and chat-based tracking.

This project strengthened my skills in prototyping, stakeholder communication, and applying UX principles in a real-world, Japanese tech setting.
